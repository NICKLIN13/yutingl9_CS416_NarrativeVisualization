Fertility Data Line Chart

Visualization Narrative

The project follows the Martini Glass narrative visualization structure. This is the final scene of the visualization story, where the user can explore reader-driven content and draw independent conclusions. A selectable line graph representing worldwide fertility is provided. By default, the line graph shows the fertility rate of the world from 1980 to 2022.

This selectable line chart is built with the D3.js library.

User Interface Events

The buttons in the navigation bar enable user interface events. Users interact by clicking on the menu items to navigate to different parts of the story. Mouse hover events in the line graph display additional data details. Clicking allows users to select a specific country and view its fertility rate over the past forty years.

Users are presented with a selection bar above the line graph, allowing them to display the fertility rate for different countries. The mouse hover event on the node point displays a tooltip containing additional information about that specific year.

Clicking on the selection bar expands the section list. Once users click on a country name, the line graph automatically refreshes and displays the new line for the selected country with a smooth animation.

Triggers

Mouse click and hover events are the triggers for this page. Clicking on the navigation buttons at the top facilitates back-and-forth navigation. Clicking also causes the map to display the fertility rate line for the selected country, removing the line of the previously selected country and showing the line for the newly selected country. Hovering over elements triggers the rendering of tooltips, presenting additional details to users.

Parameters

The fertility rate data is read from CSV.
On page load, the line graph shows the data of the world's average fertility by default. If users click on a country in selection list, javascript function receives "Country_Name" and "Country_Code" as input paramters, and refreshes line graph with the data for the requested country.
